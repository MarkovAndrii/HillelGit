
git stash   - Спрятать изменения c отметкой: sha-1 + commit
git stash save {name}   - Спрятать изменения с отметкой: {name}
git stash list  - Список спрятанных изменений
git stash apply stash@{n}   - Восстановить последние/выбранные изменения
git stash drop stash@{n}    - Удалит спрятанные изменения из тайника
git stash pop stash@{n}     - Восстановит и удалит спрятанные изменения из тайника
--index     - (при добавлении) Восстановит индексацию файлов
